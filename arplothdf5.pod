=head1 NAME

arplothdf5 - Plot all data in an HDF5 file generated by arget

=head1 SYNOPSIS

arplothdf5 [options] <hdf file[:path]>

=head1 DESCRIPTION

This utility displays a time plot of all PVs stored in the given
HDF5 data file (produced by the arget utility).
It plots PV value vs. time with one line per PV.
For waveform PVs the first point is used.

In addition to those keyboard shortcuts provided by matplotlib, two other is available.

Pressing 'y' will print a table of the PV values of all traces at the X axis position of the mouse cursor.

Pressing 'p' will print a list of value changes within the visible region.
Changes will be printed in time order to approximate the output of the camonitor utility.
